Módulo 5 - JavaScript Básico

Dê interatividade ao seu site com JavaScript.

O que é JavaScript (JS)?


JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas nas páginas web. É a "terceira camada" da web (após HTML e CSS) e é responsável por tornar as páginas **interativas**.


Como o JS Funciona no Navegador:


O JavaScript manipula o **DOM (Document Object Model)**, que é a representação em árvore da sua página HTML, permitindo que o código altere o conteúdo, a estrutura e o estilo de qualquer elemento na página em tempo real.


Conceitos Fundamentais:


  • **Variáveis**: Usamos `let`, `const` e `var` para armazenar dados.
  • **Funções**: Blocos de código reutilizáveis que executam tarefas.
  • **Eventos**: Ações do usuário ou do navegador que disparam o código (ex: `click`, `mouseover`).

Exemplo Básico de JavaScript (Alerta ao Clicar):


          
// Função simples que exibe uma mensagem
function saudacao() {
  alert("Olá, bem-vindo ao JavaScript!");
}

// Para usar em um botão HTML: <button onclick="saudacao()">Clique</button>

Exercício Prático: Crie um botão e use JavaScript para que, ao ser clicado, ele mude a cor de fundo de um parágrafo HTML que você criou anteriormente.
