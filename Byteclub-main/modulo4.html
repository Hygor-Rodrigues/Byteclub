Estilize páginas com CSS e aprenda design web responsivo.

O que é CSS?


CSS (**Cascading Style Sheets**) é a linguagem usada para descrever a **apresentação** de um documento escrito em HTML. Ele controla cores, fontes, layouts e o posicionamento dos elementos.


Sintaxe do CSS:


As regras CSS consistem em um **seletor** e um bloco de **declarações**:

          
seletor {
  propriedade: valor;
  propriedade-2: valor-2;
}
  • **Seletor**: Aponta para o elemento HTML.
  • **Declaração**: Combinação de **propriedade** e **valor**.

Exemplo Básico de CSS:


          
h1 {
  color: #333333;
  text-align: center;
}

.card {
  border: 1px solid #ccc;
  padding: 20px;
}

Exercício Prático: No arquivo `estilizacao.css`, crie uma regra que mude a cor de fundo do `` para um cinza claro e defina a fonte de todo o texto como 'Arial'.
