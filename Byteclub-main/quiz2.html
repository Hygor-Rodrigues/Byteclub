Início Fórum Suporte

1. Com base no algoritmo abaixo:

ler numero
se numero % 2 == 0 entao
escrever "Par"
senao
escrever "Ímpar"


Se for digitado o número 27, o que o algoritmo exibirá?

2. Com base no algoritmo:

total ← 0
para i de 1 até 5 faça
   total ← total + i
fimpara

Qual valor será exibido ao final?

3. Observe o algoritmo abaixo:

ler a, b, c
maior ← a
se b > maior então maior ← b fimse
se c > maior então maior ← c fimse

Se os valores forem 8, 3 e 11, qual será exibido?

4. Sobre o algoritmo:

n ← 4
enquanto n >= 0 faça
   escrever n
   n ← n - 1
fimenquanto

Qual sequência será exibida?

5. Algoritmo:

ler n
resultado ← n × 3
escrever resultado

Se n = 6, qual valor será exibido?
