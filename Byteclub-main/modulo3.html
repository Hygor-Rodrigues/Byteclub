Módulo 3 - HTML Básico

Aprenda a criar a estrutura de sites com HTML.

O que é HTML?


HTML (**HyperText Markup Language**) é a linguagem de marcação padrão usada para criar páginas web. Ele define a **estrutura** e o **conteúdo** de uma página usando um sistema de **elementos** (tags).


Estrutura de um Documento HTML:


Todo documento HTML segue uma estrutura básica. As tags principais são:

  • `<!DOCTYPE html>`: Declara o tipo de documento.
  • `<html>`: O elemento raiz que envolve todo o conteúdo da página.
  • `<head>`: Contém metadados (título da aba, links para CSS, etc.).
  • `<body>`: Contém todo o conteúdo visível da página (textos, imagens, links, etc.).

Tags Comuns e Funções:


  1. `<h1>` a `<h6>`: Títulos e subtítulos (hierarquia de conteúdo).
  2. `<p>`: Parágrafos.
  3. `<a>`: Links (âncoras). O atributo `href` define o destino.
  4. `<img>`: Inserir imagens. O atributo `src` define o caminho da imagem.
  5. `<ul>` e `<ol>`: Listas não ordenadas e ordenadas.

Exemplo Básico de HTML (Corrigido):


          
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <title>Minha Primeira Página</title>
</head>
<body>
  <h1>Bem-vindo!</h1>
  <p>Este é um parágrafo de exemplo.</p>
</body>
</html>

Exercício Prático: Crie um arquivo HTML que contenha um título principal (`um parágrafo`), (`e um link para um site de sua preferência.`)
