O Conceito de Algoritmos


Um algoritmo é uma **sequência finita de passos bem definidos e não ambíguos** que, quando executados, resolvem um problema ou realizam uma tarefa específica. Pense nele como uma receita de bolo para o computador.


Elementos Estruturais de um Algoritmo:


  • “**Entrada (Input)**”: Dados necessários para iniciar o processo.
  • “**Processamento**”: As instruções e cálculos realizados.
  • “**Saída (Output)**”: O resultado final esperado após a execução.
  • **Variáveis e Tipos de Dados**: Como armazenar e manipular a informação (números, textos, booleanos).

Formas de Representação (Pseudocódigo):


  1. **Passo a Passo (Linguagem Natural)**: Descrição simples das ações.
  2. **Fluxograma**: Representação gráfica dos passos e decisões.
  3. **Pseudocódigo**: Utiliza uma linguagem intermediária, mais próxima da programação.

Exemplo Prático (Algoritmo de Soma):


  1. **Início**.
  2. **Ler** o primeiro número (Variável A).
  3. **Ler** o segundo número (Variável B).
  4. **Calcular** a Soma = A + B.
  5. **Exibir** o valor de Soma.
  6. **Fim**.

Exercício Mental: Crie um algoritmo para verificar se um aluno foi aprovado ou reprovado (média maior ou igual a 6 é aprovação).
