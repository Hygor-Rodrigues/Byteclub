Módulo 1 - Lógica de Programação

Aprenda os fundamentos do pensamento lógico aplicado à programação

O que é Lógica de Programação?


A lógica de programação é a organização coesa de uma sequência de instruções voltadas à resolução de um problema, ou à criação de um software ou aplicação.


Conceitos Fundamentais:


  • “Algoritmo”: sequência lógica de passos para resolver um problema
  • “Variáveis”: espaços na memória para armazenar dados
  • Estruturas de Controle: if/else, laços, etc.

Exemplo Prático:


  1. Pegar o pão
  2. Passar manteiga (se houver)
  3. Adicionar recheio
  4. Fechar o sanduíche
  5. Servir

Exercício Mental: Crie um algoritmo que descreva como escovar os dentes.
